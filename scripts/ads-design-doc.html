1. Company Overview

Sathvam Natural Products is a cold-pressed oil manufacturing company headquartered in Karur, Tamil Nadu, India. We manufacture and sell 100% pure cold-pressed oils (groundnut, sesame, coconut), millets, organic spices, and natural products directly to consumers across India through our e-commerce website sathvam.in.

All products are manufactured in-house at our factory in Karur. We sell factory-direct with no intermediaries. Our primary markets are Tamil Nadu, Karnataka (Bangalore), Telangana (Hyderabad), Kerala, and pan India.

2. Purpose of the Google Ads API Tool

We are building an internal campaign management module integrated into our existing business ERP and e-commerce admin panel (admin.sathvam.in). This tool allows our marketing team to create, monitor, pause, and report on our own Google Ads campaigns without switching between multiple platforms.

The tool is used exclusively for managing Sathvam's own Google Ads account. It does not manage accounts for any third-party clients or external businesses.

3.2 System Flow

StepDescription
1Admin user logs into admin.sathvam.in using secure JWT cookie authentication
2Admin navigates to the "Google Ads" tab in the marketing section
3Frontend calls our backend API (api.sathvam.in/api/google-ads/*)
4Backend uses stored OAuth2 refresh token to get a fresh access token from Google
5Backend makes authenticated REST API calls to Google Ads API v18
6Results (campaigns, metrics, keywords) are returned to the admin panel

6. Access Control & Security

  • Who has access: Internal employees only (admin and CEO roles in our system)
  • Authentication: All API routes require a valid admin JWT cookie — no public access
  • OAuth2 tokens: Stored securely in server environment variables, never exposed to frontend
  • HTTPS only: All communication is encrypted via TLS (Let's Encrypt certificates)
  • No third-party access: The tool is not shared with or accessible by any external clients or agencies
  • Rate limiting: Backend enforces rate limits on all API routes

8. Data Handling

  • No customer PII (personal data) is collected or stored via the Google Ads API
  • Only campaign performance metrics (impressions, clicks, cost) are stored for reporting
  • No remarketing lists or customer match data is used
  • No App Conversion Tracking or mobile app campaigns are planned

9. Developer Token Usage

The developer token is used exclusively to manage Sathvam Natural Products' own Google Ads account. It is not shared with any third parties, agencies, or external clients. The token is stored as a server-side environment variable and is never exposed in frontend code or public repositories.
